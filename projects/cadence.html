Cadence measurement with a tap of a finger

Enter your athletes' names into the app. When you're ready to start a workout, select an athlete to measure and tap the screen with each stroke or step. Cadence Coach calculates the average cadence with each tap, for as long as you decide, and updates the display to show you the current average. Select another athlete and repeat. When the workout is done, select the results list to see the latest averages for all athletes that have been measured. Results are displayed as both seconds per stroke and revolutions per minute.
